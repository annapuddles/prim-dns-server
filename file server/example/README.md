# example prim-dns file server

This is an example setup of a prim-dns file server.

The directory structure seen here would actually be simulated by naming the notecards as follows:
- `/agents/index.js`
- `/agents/index.xhtml`
- `/index.xhtml`
- `/javascript/index.js`
- `/javascript/index.xhtml`
